Cable Modem Monitor

Home Assistant 的 DOCSIS 信号监控。

10+ 调制解调器 500+ 测试 6 种语言

功能

直接从电缆调制解调器的Web界面提取信号指标，并将其作为 Home Assistant 传感器公开。跟踪所有信道的下行功率、上行功率、SNR和错误计数——带有历史图表和警报。

安装

可在 HACS（Home Assistant Community Store）中获取。一键安装，自动更新。

或从 GitHub 手动安装。
